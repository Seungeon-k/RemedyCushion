# MyThermometer
Arduino, Android - 체온계

## 핀
* HC-06 <br>
VCC - 5V <br>
GND - GND <br>
TX - D2 <br>
RX - D3 <br>
 
* mlx90614 <br>
VCC - 5V <br>
GND - GND <br>
SDA - A5 <br>
SCL - A4 <br>
 <br>

## 연결
블루투스 -> hc-06 연결 (비밀번호 1234) -> 앱 -> 연결버튼 -> hc - 06 -> 연결됐다고 토스트뜸 -> 끝 <br>

연결버튼을 한번 더 누르면 연결 해제된다. (블루투스 모듈이 깜빡깜빡 빛나는 상태가 됨) <br>
 <br>
